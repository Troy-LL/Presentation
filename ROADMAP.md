# Project Roadmap

This document outlines the planned features and improvements for the Interactive Presentation App.

## 🚀 Future Features (Nice-to-Haves)

### Enhanced Audience Experience
- [ ] **Prompt Transition Alert**: Automatically notify/vibrate audience devices when the host switches to a new interaction.
- [ ] **Image/GIF Display Mode**: Allow hosts to broadcast memes or diagrams directly to audience screens.
- [ ] **Word Cloud**: Generate a live word cloud from open text responses.
- [ ] **Slider Interaction**: A "Rate this on a scale of 1–5" slider for quick sentiment checks.
- [ ] **PWA Support**: Enable "Install to Home Screen" for a native-like experience for the audience.

### Host Power Tools
- [ ] **Branded Sessions**: Custom colors, themes, and logos for the lobby and interaction screens.
- [ ] **Presentation Extension**: A browser extension that overlays host controls on top of other presentation tools (like Google Slides or Canva).
- [ ] **Preset Collections**: Group presets into "Lessons" or "Meetings" for easier organization.
- [ ] **Live Audio Streaming**: Small, low-latency audio broadcast for accessibility.

### Technical Improvements
- [ ] **Persistence Layer**: Optional database support (e.g., PostgreSQL/Vercel Postgres) for long-term session history and analytics.
- [ ] **Advanced Security**: Optional password protection for rooms or host dashboard.
- [ ] **Internationalization (i18n)**: Support for multiple languages in both the host and audience UI.

## ✅ Completed in v0.1.0
- [x] Real-time PDF slide synchronization.
- [x] Voice-activated host commands.
- [x] Multi-device host dashboard support.
- [x] Live Polls, Quizzes, Open Text, and Emoji Reactions.
- [x] Client-side PDF and CSV metrics export.
- [x] Local LAN hosting mode with auto-IP detection.
- [x] Docker & Docker Compose support.
